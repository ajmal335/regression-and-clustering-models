# Regression and Clustering Models

This repository contains regression and clustering machine learning models implemented using Python and Scikit-learn.

Models included:

• Linear Regression
• K-Means Clustering

Datasets used:

• California Housing Dataset
• Iris Dataset (for clustering visualization)

Steps performed:

• Data preprocessing
• Feature selection
• Model training
• Model evaluation
• Data visualization

Tools and Libraries used:

• Python
• Pandas
• NumPy
• Scikit-learn
• Matplotlib
• Seaborn

Purpose of this repository:

This project demonstrates practical implementation of regression and clustering techniques for structured datasets and supports understanding of supervised and unsupervised learning workflows used in real-world AI and machine learning pipelines.
